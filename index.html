How it works

Under the hood, the playground is a tiny SQL engine written entirely in JavaScript.

1
Load a CSV

The app parses your CSV into an in-memory table (list of rows) using the header row as column names. You can bring your own file or choose a sample dataset like people or sales.

2
Parse & plan

A lightweight SQL parser recognizes SELECT, FROM, WHERE, GROUP BY, and ORDER BY. It builds a small query object that describes filters, projections, aggregates, and grouping.

3
Execute in memory

The engine applies the plan: filter rows, group them, compute aggregates, sort the result, and render it as a table, JSON, CSV text, and a bar chart when possible.
